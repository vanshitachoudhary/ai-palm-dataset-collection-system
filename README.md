# AI Palm Dataset Collection System

A professional AI dataset collection web application built using React.js for capturing and organizing palm images for AI research workflows.

---

## Features

- Webcam-based palm image capture
- Structured left/right palm labeling
- Front and back camera workflow
- Participant information form
- Dataset progress tracking
- Image preview gallery
- Delete image functionality
- Dataset export support
- Professional AI-focused UI

---

## Tech Stack

- React.js
- JavaScript
- HTML5
- CSS3
- Webcam API

---

## Project Objective

This project was developed for AI research-oriented palm dataset collection workflows. The system helps in capturing structured palm images with proper labeling and participant metadata management.
